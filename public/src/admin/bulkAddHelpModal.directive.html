Bulk Add
Used to add large amounts of new words at the same time
Options
"Comma Separated Values(CSV)", "Tab Separated Values (TSV)"
"Don't add if already exists", "Replace Existing Words", "Send error upon duplicate"
"English","Orcish","Part Of Speech"
"Orcish","Part Of Speech","English"
Nouns, Verbs and Adjectives
Words of these types have their conjugations/cases generated automatically
Pronouns, Possessives, Demonstratives and Relatives
Since words of these types cannot have their cases generated automatically, an error will occur if you try to add them using bulk add
Instead please use the New Word function on the admin page
Comma Separated Values CSV
Each word is stored on a single line in the selected format. The available formats are:
"English","Orcish","Part of Speech"
"Orcish","Part Of Speech","English"
Quotes are optional if the value doesn't contain any commas, e.g. kill,ka,verb is valid, but
active,"sistrad, -ul",adjective requires quotes around "sistrad, -ul" because it has a comma
Most spreadsheet software can save their data in CSV format
If you have a CSV file, you can drag the file onto the text area or manually copy and paste its content
Tab Separated Values (TSV)
Each word is stored on a single line with a tab character sepataring each value
e.g. english orcish part of speech
When you copy cells from excel or a similar spreadsheet software and paste them into the text area, the resulting text is in TSV format
Since manually inserting tab characters is difficult, use CSV when manually inserting data, and only use TSV when copying directly from excel
Don't add if already exists
If a word with the same Orcish and Part of Speech already exists, it will not be added.
Replace existing words
All words with the same Orcish and Part of Speech as a word to be added is first removed. This will result in information such as Extra Info and Related Words being lost for the replaced words
Allow duplication
All words are added, regardless of if a word with the same Orcish and Part of Speech exists. Can result in unwanted duplication of words
