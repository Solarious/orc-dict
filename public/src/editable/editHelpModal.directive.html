Extra Keywords
Keywords are used to assist the searching algorithm in finding matches
The server automatically generates keywords for the orcish, english, and conjugations/cases/etc behind the scenes
However, when generating keywords for english, the server only generates one keyword for the entire english value, spaces included.
For example, a word with english "to wonder" will only match with "to wonder" in quotes, and will not match "to" or "wonder"
By adding Extra Keywords you can have it match "wonder", "wondered" and "wondering"
Extra Keywords: Keyword
The text this keyword will match to
Extra Keywords: Priority
Matches found from a keyword are sorted by the keyword's priority, lowest values first
Automatically generated keywords have priority 1 for orcish and english, priority 2 for conjugations and cases, and priority 3 for verb participles and agents
Extra Keywords: Type
How the keyword relates to this word e.g. "present active 1st person singular", "english" or "accusative plural"
